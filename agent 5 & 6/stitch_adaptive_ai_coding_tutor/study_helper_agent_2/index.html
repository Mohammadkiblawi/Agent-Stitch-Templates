How does recursion work again?

Imagine a set of Russian nesting dolls. Each doll contains a smaller version of itself until you reach the tiniest one. In code, it's a function calling itself until it hits a simple "base case" that stops the loop. Let's look at an example… 

def factorial(n):
    if n == 1:          # base case
        return 1
    return n * factorial(n-1)
